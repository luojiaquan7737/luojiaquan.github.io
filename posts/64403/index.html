关于博客版本区别说明

发表于 2017-10-21 |

博客被分为两个版本，分别为国际版和国内版

下面分别说明两者的差别！

国际版

评论插件：DISQUS

该插件是国外很多博客使用的评论插件，当然我也不能免俗！

搭载平台： Github pages

Github 平台处于美国地域，大陆用户时常有时候会出现打不开的时候。

国内版

评论插件： 畅言

搭载平台：Coding pages

Coding 平台处于大陆地域，也算是Github平台的分身

如何区分国内版还是国际版？

电脑：博客左上角有写国际版，那就是国际版。否则就是国内版。
手机：博客首页上方写有国内版，那就是国内版。否则就是国际版。

点击这里直达国内版博客

点击这里直达国际版博客

总结

无论博客是国际版还是国内版，博客内容实际还是一样的，就是评论和一些小部分不一样而已！也可以算是热备份，即使有一个网站访问不了，也可以瞬间转移到可以正常访问的网站上！尤其对以后的升级迁移网站的时候。

PS：文案不是太好，所以各位游客、粉丝多多包容啦！我会慢慢改进的，也会坚持每个星期写一些对大家有帮助的教程以及一些美图资源！
